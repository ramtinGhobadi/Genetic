توضیحات الگوریتم:
کروموزوم‌ها: در مثال اول به صورت باینری (رشته‌ای از 0 و 1) هستند.

تابع Fitness: معیار خوب بودن هر جواب را مشخص می‌کند.

انتخاب (Selection): روش رولت (بر اساس مقدار fitness).

ترکیب (Crossover): یک نقطه برش برای تبادل ژن بین والدین.

جهش (Mutation): معکوس کردن بیت‌ها با احتمال کم.

Elitism: حفظ بهترین جواب‌ها در هر نسل.
